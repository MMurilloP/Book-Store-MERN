# MERN BOOK-STORE

Este es un proyecto MERN stack (MongoDB, Express, React, Node.js) que trata sobre los siguientes temas:

- CRUD en el Backend
- Enrutamiento en el Backend
- Política CORS
- Operaciones en MongoDB
- CRUD en el Frontend
- Enrutamiento en el Frontend
